# 纸飞机冲榜

基于 PRD 实现的网页端轻休闲纸飞机投掷游戏。玩家可以选择纸张、调整折纸参数、把握释放时机，并通过飞行距离冲击排行榜。

## 已实现内容

- 3 种纸张底板：普通打印纸、轻薄笔记纸、硬质卡纸
- 4 个核心折纸参数：机头尖锐度、机翼宽度、左右对称度、尾翼上翘角度
- 鼠标按住蓄力、向上拖动调角度、释放区判定稳定度
- Canvas 飞行演出、距离和高度实时显示、轻量风扰动
- 结算复盘标签、个人最佳、最近成绩和本地排行榜
- GitHub Pages 自动部署工作流

## 本地启动

这是一个纯静态项目，不依赖构建工具。你可以直接使用任意静态服务器启动：

```powershell
python -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## 部署说明

仓库已配置 GitHub Pages 工作流。推送到 `main` 分支后，GitHub Actions 会自动部署站点。

说明：

- 当前“总榜”为浏览器本地可玩的 MVP 版本，会合并预置榜单和当前浏览器的本地成绩
- 如果后续要做真实全服排行，可以把成绩上传和校验切到后端服务
